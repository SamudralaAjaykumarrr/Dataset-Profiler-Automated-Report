# Dataset Profiler + Automated Report

A utility to automatically scan datasets, summarize key statistics, and generate data quality reports.

Includes:
- missing values analysis
- data type summary
- basic descriptive statistics
- automated profiling report

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the notebook or `python dataset_profiler_report.py`

## Tech Used
Python, Pandas, NumPy
